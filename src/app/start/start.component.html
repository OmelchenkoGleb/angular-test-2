Цей додаток призначений для початкового педагогічного навантаження.

Перш за все його мета це полегшення процессу розподілення предметів по викладачам та розрахунок даних на базі основних.

Під час роботи використуваються файли лише формату .xls

Додаток може розпарсити лише файли формату .xls та певної структури. Та формує для кожного викладача файл на базі шаблону, якій також завантажується в систему.

Формування звіту

Під час обробки даних необхідно сформувати звіт по всім наявним даним в системі для кожного викладача. Цей звіт формується та його можна переглянути.
